The Unmapped Concepts module evaluates the distribution of unmapped values in a
cohort, looking both at the proportion of unmapped values for a given variable
and at the median number of unmapped rows per patient in the cohort.

## Installation

You can install the development version of this package like so:

``` r
devtools::install_github('ssdqa/unmappedconcepts')
```
